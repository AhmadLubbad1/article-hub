# 📰 ArticleHub

A full-stack Article Management System where users can publish, browse, 
and manage articles — with role-based access for admins, authors, and readers.

## 🚀 Features

- 🔐 User Authentication — secure register & login with session management
- 👥 Role-Based Access — Admin, Author, and User roles with different permissions
- 📝 Article CRUD — create, edit, delete, and read articles with featured images
- 📋 Draft & Publish — authors can save drafts and publish when ready
- 🔍 Search & Filter — search articles and filter by category
- 📱 Responsive Design — mobile-friendly UI built with Bootstrap
- 🛠️ Admin Dashboard — manage users and all content from one place

## 🛠️ Built With

- **Frontend:** HTML, CSS, JavaScript, Bootstrap
- **Backend:** PHP
- **Database:** MySQL
- **Data Formats:** JSON, XML

## ⚙️ How to Run

1. Clone the repo
2. Import the database file from the `/database` folder into phpMyAdmin
3. Configure `/includes/config.php` with your DB credentials
4. Run on a local server (XAMPP or WAMP)
5. Open `http://localhost/ArticleHub`
